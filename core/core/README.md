# OFN Core Identity

This directory defines the immutable identity of the
Open Financial Network (OFN).

Anything placed here is considered:
- Normative
- Canonical
- Governance-protected

## What belongs here
- Definitions
- Principles
- Guarantees
- Non-goals

## What does NOT belong here
- Code
- Implementations
- Experiments
- Business logic

Changes to this directory require:
- Formal RFC
- Versioned release
- Explicit approval
